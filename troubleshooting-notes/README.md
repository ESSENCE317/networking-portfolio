# Troubleshooting Notes & Scenarios

## Networking Issues
- Incorrect subnet mask breaking connectivity
- Default gateway misconfiguration
- Devices connected to LAN but no internet access
- DNS resolution failures
- IPv4 vs IPv6 routing confusion

## Tools Used
- ping
- traceroute
- arp
- Wireshark
- Cisco IOS show commands

## Mindset
Focused on methodical diagnosis:
1. Physical layer checks
2. IP configuration verification
3. Gateway and routing validation
4. DNS and application-layer testing
